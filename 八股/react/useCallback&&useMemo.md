## useCallback 详解

### 工作原理

useCallback 返回一个**记忆化的回调函数**，只有当依赖项发生变化时，才会创建新的函数引用。

## useMemo 详解

## 区别对比

| 特性         | useCallback            | useMemo              |
| ------------ | ---------------------- | -------------------- |
| **返回类型** | 函数                   | 任意值               |
| **主要用途** | 缓存函数引用           | 缓存计算结果         |
| **适用场景** | 传递给子组件的回调函数 | 昂贵的计算、对象创建 |
| **性能影响** | 减少子组件重新渲染     | 避免重复计算         |
| **依赖项**   | 函数内部使用的变量     | 计算所需的变量       |
